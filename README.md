# Linux Learning Platform

Dieses Projekt kombiniert mehrere Linux- und Git-Aufgaben in einem Repository.

## Inhalte
- Linux-Wissenswiki in Markdown
- Bash-Skript-Sammlung
- Monitoring-Dashboard mit Bash
- Firmenserver-Simulation
- Linux-Lernplattform mit Lektionen und Übungen

## Lernziele
- Git verwenden: clone, add, commit, push, pull
- Branches erstellen und zusammenführen
- Merge-Konflikte erkennen und lösen
- Markdown-Dokumentation schreiben
- Bash-Skripte entwickeln und strukturieren
