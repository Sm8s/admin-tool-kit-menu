# Merge-Konflikt Demo

Ein Merge-Konflikt entsteht, wenn zwei Branches denselben Bereich einer Datei unterschiedlich ändern und Git die Unterschiede nicht automatisch zusammenführen kann. [web:10][web:58]

## Beispiel
```text
Version aus deinem Branch
Version aus anderem Branch
```

Diese Markierungen müssen manuell bearbeitet und danach neu committet werden. [web:55][web:58]
